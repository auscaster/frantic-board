# Contract Drafter

A Solidity smart contract for drafting, signing, and finalizing agreements between two parties. Each agreement is uniquely identified and stores the content hash of the terms. Both parties must sign before finalization.

## Usage

1. Install dependencies: `npm install`
2. Compile: `npx hardhat compile`
3. Test: `npx hardhat test`

## Deploy

Use Hardhat to deploy to any EVM-compatible network.
