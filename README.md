# Expense Copilot (RunX Skill)

A RunX skill to manage personal expenses. Track, categorize, and summarize your spending with simple voice or text commands.

## Features

- Add an expense with description, amount, and category
- List all expenses
- Get total spending by category
- Clear all expenses

## Usage

1. Install dependencies: `pip install -r requirements.txt`
2. Run the skill: `python skill.py`
3. Interact via RunX interface (e.g., "Add expense for coffee $5 Food")

## Commands

- `Add an expense`: description, amount, category (optional, defaults to General)
- `List expenses`: show all entries
- `Get total by category`: summary by category
- `Clear all expenses`: reset the expense list
